apt install pyhton3-pip

Python csomag: virtualenv


sudo pip install virtualenv
Virtualenv: elszeparált környezet az adott fejlesztéshez... Lértehozása:
virtualenv -p pyhton3 /home/userem/virtualenv/sajat_appom
A virtualenv-et létrehozás után aktiválni kell:
source /home/userem/virtualenv/sajat_appom/bin/activate
Ezután minden amit pythonban csinálunk, a virtualenv-en belül fog működni CSAK.
